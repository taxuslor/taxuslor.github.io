和弦
2023/7/29

和弦

和弦

  • 在多声部音乐中，可以按照三度关系排列起来的三分以上的音的结合，叫做和弦

三和弦

  • 由三个音按照三度关系叠置起来的和弦，叫做三和弦

  • 在三和弦中，下面的音叫做根音，或一度音，用数字1来代表；
    中间的音叫做三度音，用数字3来代表；
    上面的音叫做五度音，用数字5来代表

  • 三和弦的类型：

    • 大三和弦：根音到三度音是大三度，三度音到五度音是小三度，根音到五度音是纯五度
    • 小三和弦：根音到三度音是小三度，三度音到五度音是大三度，根音到五度音是纯五度
    • 增三和弦：根音到三度音和三度音到五度音都是大三度，根音到五度音是增五度
    • 减三和弦：根音到三度音和三度音到五度音都是小三度，根音到五度音是减五度

七和弦

  • 由四个音按照三度关系叠置起来的和弦，叫做七和弦
  • 在四和弦中，第四个音叫做七度音，用数字7来代表
  • 七和弦的类型：
    • 以大三和弦为基础，根音至七度音为小七度的七和弦叫做大小七和弦（大调小七和弦）
    • 以小三和弦为基础，根音至七度音为小七度的七和弦叫做小小七和弦（小七和弦）
    • 以减三和弦为基础，根音至七度音为小七度的七和弦叫做减小七和弦（半减七和弦）
    • 以减三和弦为基础，根音至七度音为减七度的七和弦叫做减减七和弦（减七和弦）

原位和弦及转位和弦

  • 原位和弦：以和弦的根音为低音的和弦
  • 转位和弦：以和弦的三度音、五度音、七度音为低音的和弦
  • 三和弦除了根音外，还有两个音，所以三和弦有两个转位
    • 以三度音为低音的三和弦叫做三和弦的第一转位，也叫六和弦，用数字6来代表
    • 以五度音为低音的三和弦叫做三和弦的第二转位，也叫做四六和弦
  • 七和弦除了根音外，还有三个音，所以七和弦有三个转位
    • 以三度音为低音的七和弦叫做七和弦的第一转位，也叫做五六和弦
    • 以五度音为低音的七和弦叫做七和弦的第二转位，也叫做三四和弦
    • 以七度音为低音的七和弦叫做七和弦的第三转位，也叫做二和弦

等和弦

  • 两个和弦孤立听起来时，具有相同的声音效果，但在音乐中的意义不同，写法不同
  • 等和弦和等音程一样是由于等音变化而产生的
  • 类别：
    • 和弦中的音不因为等音变化而更改音程的结构
    • 由于等音变化而更改和弦的结构

调式总论

调式 音阶 调 调性

  • 调式：按照一定关系连结在一起的许多音，组成一个体系，并以一个音为中心（主音）
  • 音阶：调式中的音，按照高低次序，由主音到主音排列起来就叫做音阶
  • 调：调是调式的音高位置。调的名称由两部分组成，即主音的标记和调式的标记（例如：以C为主音的宫调式叫做C宫调式，以C为主音的大调式叫做C大调式）
  • 调性：调本身所具有的特质

调式音级及其特性

  • 稳定音：在调式体系中，起着支柱作用并给人以稳定感的音，即为稳定音
  • 主音：在稳定音中，最稳定的具有中心作用的音
  • 倾向：不稳定音具有进行到稳定音的特性
  • 解决：不稳定音根据其倾向进行到稳定音

多声部音乐中的调式关系

  • 在多声部音乐中，各声部纵的关系形成和弦，因此在多声部音乐中，起稳定作用的就不仅是某一个音，而是整个的和弦。在多声部音乐中，通常以主音上的大小三和弦为稳定和弦。

记谱法
2023/7/26

记谱法

音符与休止符

  • 可参考 Sight Reading 101 Notes and Rest Values

音符的写法

  • 音符包括三个组成部分：符头（空心的或实心的椭圆形标记）、符干（垂直的短线）和符尾（连在符干一端的旗状标记）
  • 音符的符头可以记在五线谱的线上与间内。符头在五线谱上的位置愈高音音愈高。
  • 用单符干记谱，符头在第三线以上时，符干朝下，写在符头的左边；在第三线以下时，符干朝上，写在符头的右边；符头在第三线上，符干朝上朝下都可以，根据相邻的符干方向而定。符尾永远写在符干的右边并弯向符头。
  • 符干的长度一般应保持八度音程的距离
  • 附点写在音符符头和休止符右边的间内

谱号

  • G谱号/高音谱号（ 𝄞 ）： 表示小字一组的g
  • F谱号/低音谱号（ 𝄢 ）： 表示小字组的f
  • C谱号（ 𝄡 ）：表示小字一组的c

增长音值的补充记号

  • 附点：是记在符头右边的小圆点，带有一个附点的音符，增长原有音符时值的二分之一；带有两个附点的音符，增长原有音符时值的四分之三
  • 延音线：用在音高相同的两个或两个以上的音符上时，表示它们要唱成一个音，它的长度等于这些音符的总和
  • 延长号：在单声部乐曲中它卸载音符和休止符的上面，表示按作品的风格、演奏者的意图可自由地增长音符和休止符的时值；记在小节线上表示小节之间的片刻休止；记在双纵线上表示乐曲的结束和告一段落。

变音记号

  • 升记号（ # ）：将基本音级升高半音
  • 降记号（ b ）：将基本音级降低半音
  • 重升记号（ x ）：将基本音级升高全音
  • 重降记号（ bb ）：将基本音级降低全音
  • 还原记号（ ♮ ）：表示将已经升高或降低的音还原

音律
2023/7/24

音律

  • 音律：各音的绝对准确高度及相互关系

十二平均律

  • 十二平均律：将八度分为十二个均等的部分（半音）的音律
    半音是十二平均律中组织中最小的音高距离
    两音间的距离等于两个半音的叫做全音

自然半音和自然全音 变化半音和变化全音

  • 自然半音：由两个相邻的音级构成的半音叫做自然半音
  • 自然全音：由两个相邻的音级构成的全音叫做自然全音
  • 变化半音：由同一音级的两种不同形式所构成的半音叫做变化半音
  • 变化全音：由同一音级的两种不同形式所构成的全音叫做变化全音

等音

  • 等音：音高相同而意义和记法不同的音

音及音高
2023/7/23

音及音高

音

  • 音是由于物体的振动而产生的
  • 音有高低、强弱、长短、音色等四种性质：
    • 高低：振动次数（频率）
    • 强弱：振幅
    • 长短：延续时间
    • 音色：发音体的性质、形状及其泛音的多少
  • 根据音的振动状态的规则与不规则，有乐音与噪音两类
  • 复合音：发音体不仅全段在振动，它的各部分也分别的同时在振动
  • 基音：发音体全段振动产生的音
  • 泛音：发音体各部分振动而产生的音

乐音体系 音列 音级

  • 乐音体系：在音乐中使用的、有固定音高的音的总和

  • 音列：乐音体系中的音，按照上行或下行次序排列起来

  • 音级：乐音体系中的各音，分为基本音级和变化音级

  • 乐音体系中，七个具有独立名称的音级叫做基本音级

    字母体系：C d e f g a b

    唱名体系：do re mi fa sol la si

    钢琴上五十二个白键循环重复地使用七个基本音级名称

  • 八度：两个相邻的具有同样名称的音

  • 变化音级：升高或降低基本音级而得来的音

    升高半音用 升 或 # 表示

    降低半音用 降 或 b 表示

    升高全音用 重升 或 x 表示

    降低全音用 重降 或 bb 表示

音的分组

  • 钢琴上在音列中央的一组叫做小字一组，音级标记用小写字母并在右上方加数字1来表示（ex: c1 d1 e1）

  • 比小字一组高的组顺次定名为：小字二组、小字三组、小字四组、小字五组

  • 比小字一组低的组依次定名为：小字组、大字组、大字一组、大字二组

  • 小字组标记使用不带数字的小写字母，大字组标记使用不带数字的大写字母

  • 大字一组用大写字母并在右下方加数字1来表示（ex: C1 D1 E1）

音域及音区

  • 音域有总的音域和个别的人声或乐器的音域
  • 总的音域是指音列的总范围，即从最低音到最高音（C2 ~ c5）的距离
  • 个别的人声或乐器的音域是指在整个音域中所能达到的那一部分，如钢琴音域为（A2 ~ c5）
  • 音区是音域的一部分，有高音区、中音区、低音区
  • 高音区：小字三组、小字四组、小字五组
    中音区：小字组、小字一组、小字二组
    低音区：大字组、大字一组、大字二组
