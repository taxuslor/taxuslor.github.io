日本語文法（三）

1.ここ、そこ、あそこ　は　名　です

  • 用于指示场所，相对位置关系与これ、それ、あれ相同

ここはデパート（でぱあと）です。　这是百货商店。

そこは図書館（としょかん）です。 那里是图书馆。

あそこは入り口（いりくち）です。　那儿是入口。

2.名　は　名[场所]　です

  • 表示“名词”存在于“名词[场所]”中

食堂（しょくどう）はデパートの七階（かい）です。　食堂在百货商店的7层。

トイレ（といれ）はここです。　厕所在这里。

小野（おの）さんは事務所（じむしょ）です。　小野女士在事务所。

3.名　は　どこですか

  • 用于询问存在的场所

トイレはどこですか。　厕所在哪里？

あちらです。　在那儿。

あなたのかばんはどこですか。　你的包在哪里？

私のかばんはここです。　我的包在这里。

4.名　も　名　です

  • “も”相当于汉语中的“也”

李さんは中国人です。　小李是中国人。

張（ちょう）さんも中国人です。　小张也是中国人。

あなたも中国人です。 你也是中国人吗？

5.名　は　名ですか、　名ですか

  • 选择疑问句

かばん売り場（うりば）は一階ですか、2階ですか。　卖包的柜台在一层还是二层？

今日（きょう）は水曜日（すいようび）ですか、木曜日（もくようび）ですか。　今天是星期三还是星期四？

6.名　は　いくらですか

  • 询问价钱

これはいくらですか。　这个多少钱？

その服（ふく）はいくらですか。　那件衣服多少钱？

日本語文法（二）
2023/8/20

日本語文法（二）

1.これ、それ、あれ　は　名　です

  • 说话人和听话人相隔一段距高， 面对面时：
    • これ：距说话人较近的事物[说话人的范围、领域内的事物]
    • それ：距听话人较近的事物[听话人的范围、领城内的事物]
    • あれ：距说话人和听话人都比较远的事物[不属于任何一方范围、领域内的事物]
  • 说话人和听话人位于同一位置，面向同一方向时:
    • 距说话人、听话人较近的事物
    • 距说话人、听话人较远的事物
    • 距说话人、听话人更远的事物

これは本（ほん）です。 这是书。

それはかばんです。 那是包。

あれはテレビ（てれび）です。 那是电视机。

2.誰ですか、何ですか

  • 分别相当于汉语的 是谁 和 是什么

それは何ですか。 那是什么？

あの人誰ですか。 那个人是谁？

  • だれ　的比较礼貌的说法是　どなた
    • だれ　通常用手対方与自己地位相当或比自己地位低时
    • 对尊长或比自己地位高的人用　どなた　询问

李さんはどなたですか。 李先生是哪一位？

3.この、その、あの　名　は　名　です

  • 修饰名词的所属关系，相对位置关系与　これ、それ、あれ　相同

このカメラ（かめら）は李さんのです。　这个照相机是李先生的。

その自転車（自転車）は森さんのです。　那辆自行车是森先生的。

あのノート（のうと）は誰のですか。　那个笔记本是谁的？

4.どれ、どの　名

  • どれ、どの　是在三个以上的事物中，不能确定是哪一个时所用的疑问词

森さんのかばんどれはですか。　森先生的包是哪个?

長島さんの傘（かさ）はどれですか。　长岛先生的伞是哪一把?

小野さんの机（つくえ）はどの机ですか。　小野女士的桌子是哪一张？

日本語文法（一）
2023/8/8

日本語文法（一）

1.名　は　名　です

  • 相当于汉语的 是 。 ~は 是主语部分， ~です 是谓语部分。助词 は 用于提示主题，读做 わ 。

私は中国人です。　我是中国人。

2.名　は　名　ではありません

  • 相当于汉语的 不是 。 ではありません 的 では ，在口语中有时会发成 じゃ 。

私は日本人では（じゃ）ありません。 我不是日本人。

3.疑问句及问答

名　は　名　ですか

  • 相当于汉语的 是吗? 。助词 か 接在句尾表示疑问。日语的问句在句尾不使用 ? 。

あなたは学生ですか。 你是学生吗？

应答

  • 回答疑问句的时候，可以只用 はい 或者 いいえ 。亦或是 はい、そうです 和 いいえ、ちがいます
  • 不知道时用 わかりません

はい、そうです。 是，是学生。

いいえ、違います。 不，不是。

分かりません。 我不知道。

4.名　の　名 「从属机构、国家」「属性」

  • 助词 の 连结名词和名词，表示前面的名词是后面名词从属的机构、国家或属性。

私の父 我父亲

5.人称

わたし　あなた　あの人

  • 第一人称　わたし　第二人称　あなた　第三人称　あの人
  • 在不知道对方的姓名而又必须打招呼的时候，才会使用第二人称 あなた ，因为用 あなた 来称呼对方，有时会显得很不礼貌。

～さん

  • 称呼别人时，不分男女，都在其姓后面加 さん 。

  • 称呼小孩的时候，一般在其名字后加 ちゃん 。

  • 对于与自己年龄相当，或比自己年轻的男性，有时也用 くん 称呼。

和弦
2023/7/29

和弦

和弦

  • 在多声部音乐中，可以按照三度关系排列起来的三分以上的音的结合，叫做和弦

三和弦

  • 由三个音按照三度关系叠置起来的和弦，叫做三和弦

  • 在三和弦中，下面的音叫做根音，或一度音，用数字1来代表；
    中间的音叫做三度音，用数字3来代表；
    上面的音叫做五度音，用数字5来代表

  • 三和弦的类型：

    • 大三和弦：根音到三度音是大三度，三度音到五度音是小三度，根音到五度音是纯五度
    • 小三和弦：根音到三度音是小三度，三度音到五度音是大三度，根音到五度音是纯五度
    • 增三和弦：根音到三度音和三度音到五度音都是大三度，根音到五度音是增五度
    • 减三和弦：根音到三度音和三度音到五度音都是小三度，根音到五度音是减五度

七和弦

  • 由四个音按照三度关系叠置起来的和弦，叫做七和弦
  • 在四和弦中，第四个音叫做七度音，用数字7来代表
  • 七和弦的类型：
    • 以大三和弦为基础，根音至七度音为小七度的七和弦叫做大小七和弦（大调小七和弦）
    • 以小三和弦为基础，根音至七度音为小七度的七和弦叫做小小七和弦（小七和弦）
    • 以减三和弦为基础，根音至七度音为小七度的七和弦叫做减小七和弦（半减七和弦）
    • 以减三和弦为基础，根音至七度音为减七度的七和弦叫做减减七和弦（减七和弦）

原位和弦及转位和弦

  • 原位和弦：以和弦的根音为低音的和弦
  • 转位和弦：以和弦的三度音、五度音、七度音为低音的和弦
  • 三和弦除了根音外，还有两个音，所以三和弦有两个转位
    • 以三度音为低音的三和弦叫做三和弦的第一转位，也叫六和弦，用数字6来代表
    • 以五度音为低音的三和弦叫做三和弦的第二转位，也叫做四六和弦
  • 七和弦除了根音外，还有三个音，所以七和弦有三个转位
    • 以三度音为低音的七和弦叫做七和弦的第一转位，也叫做五六和弦
    • 以五度音为低音的七和弦叫做七和弦的第二转位，也叫做三四和弦
    • 以七度音为低音的七和弦叫做七和弦的第三转位，也叫做二和弦

等和弦

  • 两个和弦孤立听起来时，具有相同的声音效果，但在音乐中的意义不同，写法不同
  • 等和弦和等音程一样是由于等音变化而产生的
  • 类别：
    • 和弦中的音不因为等音变化而更改音程的结构
    • 由于等音变化而更改和弦的结构

调式总论

调式 音阶 调 调性

  • 调式：按照一定关系连结在一起的许多音，组成一个体系，并以一个音为中心（主音）
  • 音阶：调式中的音，按照高低次序，由主音到主音排列起来就叫做音阶
  • 调：调是调式的音高位置。调的名称由两部分组成，即主音的标记和调式的标记（例如：以C为主音的宫调式叫做C宫调式，以C为主音的大调式叫做C大调式）
  • 调性：调本身所具有的特质

调式音级及其特性

  • 稳定音：在调式体系中，起着支柱作用并给人以稳定感的音，即为稳定音
  • 主音：在稳定音中，最稳定的具有中心作用的音
  • 倾向：不稳定音具有进行到稳定音的特性
  • 解决：不稳定音根据其倾向进行到稳定音

多声部音乐中的调式关系

  • 在多声部音乐中，各声部纵的关系形成和弦，因此在多声部音乐中，起稳定作用的就不仅是某一个音，而是整个的和弦。在多声部音乐中，通常以主音上的大小三和弦为稳定和弦。

记谱法
2023/7/26

记谱法

音符与休止符

  • 可参考 Sight Reading 101 Notes and Rest Values

音符的写法

  • 音符包括三个组成部分：符头（空心的或实心的椭圆形标记）、符干（垂直的短线）和符尾（连在符干一端的旗状标记）
  • 音符的符头可以记在五线谱的线上与间内。符头在五线谱上的位置愈高音音愈高。
  • 用单符干记谱，符头在第三线以上时，符干朝下，写在符头的左边；在第三线以下时，符干朝上，写在符头的右边；符头在第三线上，符干朝上朝下都可以，根据相邻的符干方向而定。符尾永远写在符干的右边并弯向符头。
  • 符干的长度一般应保持八度音程的距离
  • 附点写在音符符头和休止符右边的间内

谱号

  • G谱号/高音谱号（ 𝄞 ）： 表示小字一组的g
  • F谱号/低音谱号（ 𝄢 ）： 表示小字组的f
  • C谱号（ 𝄡 ）：表示小字一组的c

增长音值的补充记号

  • 附点：是记在符头右边的小圆点，带有一个附点的音符，增长原有音符时值的二分之一；带有两个附点的音符，增长原有音符时值的四分之三
  • 延音线：用在音高相同的两个或两个以上的音符上时，表示它们要唱成一个音，它的长度等于这些音符的总和
  • 延长号：在单声部乐曲中它卸载音符和休止符的上面，表示按作品的风格、演奏者的意图可自由地增长音符和休止符的时值；记在小节线上表示小节之间的片刻休止；记在双纵线上表示乐曲的结束和告一段落。

变音记号

  • 升记号（ # ）：将基本音级升高半音
  • 降记号（ b ）：将基本音级降低半音
  • 重升记号（ x ）：将基本音级升高全音
  • 重降记号（ bb ）：将基本音级降低全音
  • 还原记号（ ♮ ）：表示将已经升高或降低的音还原

音律

  • 音律：各音的绝对准确高度及相互关系

十二平均律

  • 十二平均律：将八度分为十二个均等的部分（半音）的音律
    半音是十二平均律中组织中最小的音高距离
    两音间的距离等于两个半音的叫做全音

自然半音和自然全音 变化半音和变化全音

  • 自然半音：由两个相邻的音级构成的半音叫做自然半音
  • 自然全音：由两个相邻的音级构成的全音叫做自然全音
  • 变化半音：由同一音级的两种不同形式所构成的半音叫做变化半音
  • 变化全音：由同一音级的两种不同形式所构成的全音叫做变化全音

等音

  • 等音：音高相同而意义和记法不同的音
