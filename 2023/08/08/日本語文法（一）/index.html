日本語文法（一）

1.名　は　名　です

  • 相当于汉语的 是 。 ~は 是主语部分， ~です 是谓语部分。助词 は 用于提示主题，读做 わ 。

私は中国人です。　我是中国人。

2.名　は　名　ではありません

  • 相当于汉语的 不是 。 ではありません 的 では ，在口语中有时会发成 じゃ 。

私は日本人では（じゃ）ありません。 我不是日本人。

3.疑问句及问答

名　は　名　ですか

  • 相当于汉语的 是吗? 。助词 か 接在句尾表示疑问。日语的问句在句尾不使用 ? 。

あなたは学生ですか。 你是学生吗？

应答

  • 回答疑问句的时候，可以只用 はい 或者 いいえ 。亦或是 はい、そうです 和 いいえ、ちがいます
  • 不知道时用 わかりません

はい、そうです。 是，是学生。

いいえ、違います。 不，不是。

分かりません。 我不知道。

4.名　の　名 「从属机构、国家」「属性」

  • 助词 の 连结名词和名词，表示前面的名词是后面名词从属的机构、国家或属性。

私の父 我父亲

5.人称

わたし　あなた　あの人

  • 第一人称　わたし　第二人称　あなた　第三人称　あの人
  • 在不知道对方的姓名而又必须打招呼的时候，才会使用第二人称 あなた ，因为用 あなた 来称呼对方，有时会显得很不礼貌。

～さん

  • 称呼别人时，不分男女，都在其姓后面加 さん 。

  • 称呼小孩的时候，一般在其名字后加 ちゃん 。

  • 对于与自己年龄相当，或比自己年轻的男性，有时也用 くん 称呼。
