2023/8/10 Overview Vim

Vim Overview

  • 本篇文章以vimtutor为大纲，更多内容参见Vim Documentation

第一讲

移动光标

​ ^

​ k

< h l >

​ j

​ v

Vim的进入与退出

:q!

放弃所有改动并退出

文本编辑之删除

x

删除光标所在位置的字符

文本编辑之插入

i

在光标前插入文本

文本编辑之添加

A

在行末添加内容

编辑文件

:wq

保存改动并退出

第二讲

删除类命令

dw

从当前光标当前位置直到下一个单词起始处，不包括它的第一个字符

d$

从当前光标当前位置直到单词末尾，包括最后一个字符

de

从当前光标当前位置直到当前行末

dd

删除整一个当前行

使用计数指定动作

operator [number] motion

重复操作一定次数操作对象

移动光标

0

移动光标到行首

撤销类命令

u

撤消以前的操作

U

撤消在一行中所做的改动

CTRL-R

撤消以前的撤消命令，恢复以前的操作结果

第三讲

置入类命令

p

将最后一次删除的内容置入光标之后

替换类命令

r

输入r和一个字符替换光标所在位置的字符

更改类命令

ce

改变文本直到一个单词的末尾

c [number] motion

改变一定数量的操作对象

c$

改变光标后的整行

第四讲

定位及文件状态

CTRL-G

显示当前编辑文件中当前光标所在行位置以及文件状态信息

G

光标跳转到文件最后一行

gg

光标跳转到文件第一行

[number] G

跳转到对应行

搜索类命令

/

输入 / 加上一个字符串可以在当前文件中查找该字符串

输入n即可正向遍历该字符串，输入N即可逆向遍历该字符串

?/

输入 ?/ 加上一个字符串可以从文件末尾开始查找该字符串

CTRL-O

CTRL-I

配对括号的查找

%

查找配对的括号 ) ] }

替换命令

:s/old/new

只将光标所在行的第一个匹配串进行替换

:s/old/new

替换全行的匹配串

:[number],[number]s/old/new/g

替换若干行间的匹配串

:%s/old/new/g

替换整个文件中的匹配串

:%s/old/new/gc

找到整个文件中的每个匹配串，并且询问是否进行替换
